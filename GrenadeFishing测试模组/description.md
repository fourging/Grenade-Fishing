# 炸鱼 Mod — 就是把手雷丢进水里炸鱼 🐟💥

## 简介

想要更暴力、更滑稽的钓鱼体验？
本 Mod 允许你用各种手雷把水面当作猎场 —— 直接炸出鱼来。
没别的隐藏功能、没复杂菜单，就是：丢手雷 → 看鱼飞。
甚至连屎蛋也能炸鱼。对，就是屎蛋。
![炸鱼Mod预览](https://images.steamusercontent.com/ugc/10429282544672176288/A6C22078B6AA35020AD296858107F0C353A7C5B9/?imw=5000&imh=5000&ima=fit&impolicy=Letterbox&imcolor=%23000000&letterbox=false)

## 主要特性

* 使用任何手雷/炸药炸鱼（只要是有触发原版爆炸逻辑功能的都行，包括屎蛋）。
* 手雷品质与价格会影响产出概率与“奇迹”爆率（高价手雷更容易出好鱼）。
* 简洁直接：不改游戏钓鱼逻辑，只在爆炸时候检测是不是水中爆炸，然后凭空生成鱼。
* 带有保底机制，避免长时间空手而归。

> **特别说明 — 屎蛋保底（关键）**
> 当你累计用「屎蛋」炸鱼超过 **114 次** 后，每次使用屎蛋都会直接产出表中价值最高的【黄金鱼】。

## 简短概率提示（供好奇玩家参考）

* 鱼类按价值分档：常见 / 普通 / 稀有 / 顶级（>6000）—— 大致基础分布约 **68% / 22% / 7% / 3%**。
* 手雷成功产出判定（按手雷价格分档位）： ≈ **2%（屎球） / 6% / 16% / 28% / 48%**（高价手雷可有小幅提升）。
* 奇迹（直接出顶级鱼）基础约 **1%**，随手雷价格上升可达约 **3%**。
* 哑雷（无产出）概率：约 **1%**。

## 注意事项 / 兼容性

* 本 Mod 只改变“手雷被不小心丢水里了”的产出逻辑，不影响角色移动/战斗等其它系统。
* 若你在使用其它对手雷或掉落物有深度修改的 Mod，可能会产生冲突。
